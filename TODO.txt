CLIENT

OK : Le client voit la liste des canaux en cours (commande LIST)
OK : Le client rejoint un canal ("channel") de communication (commande JOIN). Si le canal n'existe pas, il est créé.
OK : Le client peut voir la liste des participants (commande WHO)
OK : Le client discute avec les autres clients connectés sur le canal (pas de commande) ou en privé avec un autre client du canal (commande PRV_MSG nick)
OK : A tout moment, le client peut quitter le canal (commande LEAVE) pour en choisir un autre (JOIN) ou se déconnecter du serveur (commande BYE. Il faut quitter le canal avant de pouvoir utiliser cette commande).

Le premier client connecté sur un canal devient l'administrateur du canal. Il doit être identifié comme tel dans la liste des participants (commande WHO, par exemple en affichant son nom de la façon suivante : @nick@). L'administrateur peut en plus utiliser les commandes suivantes:
OK : KICK nick : sort le client concerné du canal (équivalent de LEAVE)
OK : KILL nick : déconnecte brutalement le client du serveur (équivalent de BYE mais sans se préoccuper du canal)
OK : BAN nick : déconnecte le client (comme KILL) et l'ajoute (son adresse IP) dans une liste de clients interdits
